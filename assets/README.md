# Assets

This directory contains the app's visual assets.

## Required Assets

### App Icons
- `icon.png` - App icon (1024x1024px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Web favicon (48x48px)

### Splash Screen
- `splash.png` - Splash screen (1242x2436px)

## Asset Guidelines

### Colors
- Primary: #1a472a (Dark Green)
- Secondary: #d4af37 (Gold)
- Background: #2a5a3a (Medium Green)

### Fonts
- Default: System fonts (San Francisco on iOS, Roboto on Android)

## Creating Assets

You can use any design tool (Figma, Sketch, Adobe XD) to create these assets.

### Icon Template
- Square image with rounded corners
- Use a spade symbol (â™ ) as the primary element
- Use the color scheme above
- Ensure good contrast for visibility

### Splash Screen Template
- Portrait orientation
- Center the app logo
- Use dark green background
- Include game name in gold text

## Generating Assets

If you have a single source image, you can use:
- [makeappicon.com](https://makeappicon.com/) for iOS/Android icons
- [appicon.co](https://appicon.co/) for complete icon sets
- Expo's built-in asset generation

## Placeholder Assets

For development, you can use simple colored squares as placeholders:
- Icon: 1024x1024 dark green square with gold "CB" text
- Splash: 1242x2436 dark green rectangle with centered gold "Call Break" text

These should be replaced with professional designs before release.
